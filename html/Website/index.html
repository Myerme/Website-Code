I created this website with the hopes of learning a new set of skills that such as web development and to gain an insight on how web security works. This page was also created to learn and test different cyber-attacks and to see how a hacker would go about stealing personal information form the site and or taking the site down. Internet security and ethical hackers have always fascinated me for as long as I can remember, and I have always wanted to try and find a way I can learn about how they do what they do. So, I found a mentor who set me the task of creating a website, and once the website was created with a way to submit data, then we would look at DDOS attacks and stealing information from the site. Although I have had no prior knowledge of HTML, CSS or PHP, I do have an in depth understanding of python. Despite not having any prior understanding I quickly managed to grasp HTML and CSS, although PHP I still need work on.
